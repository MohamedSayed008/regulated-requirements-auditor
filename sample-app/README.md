# sample-tenancy-manager

A small, deliberately imperfect tenancy-management module used as the audit
target for Mizan. It models the rent, notice, eviction, registration, and
deposit logic a Dubai property-management app would contain.

**This code contains intentional compliance violations against Dubai tenancy
law.** They are seeded so the auditor's precision and recall can be measured
against a known ground truth. Do not use this as a reference implementation.
