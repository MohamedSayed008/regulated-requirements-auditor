# sample-einvoicing-app

A small, deliberately imperfect UAE invoicing module used as the audit target
for the eInvoicing corpus. It models the build, format, transmit, and storage
logic a billing system would contain.

**This code contains intentional compliance violations against the UAE
eInvoicing mandate.** They are seeded so the auditor's precision and recall can
be measured against a known ground truth. Do not use this as a reference
implementation.
